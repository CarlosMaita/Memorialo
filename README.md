# Memorialo
Memorialo is a project designed for events providers and users who want to hire event services.  
In memorialo, we make easier how the people hire a event service. we offer contrants, chat, notifications and providers at the same place. 

## files 
- `Docs` - this is the file where is located all the documentation of memorialo.
- `figma` - this is the file where is located the frontend app
- `laravel ` - this is the file where is located the backend app 

## Branches
- `main` -  this is the main branch deployed in production server.
- `stage`  - this is the dev branch deployed in testing server.
